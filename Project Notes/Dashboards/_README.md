# Dashboards

Use this folder for Obsidian Bases and generated indexes that make the notes system easier to review.

Dashboards should surface operational questions, not duplicate every note:

- What is active, blocked, stale, or unverified?
- Which known-good notes need revalidation?
- Which releases or incidents have incomplete evidence?
